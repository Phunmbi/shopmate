## Shopmate 
An e-commerce desktop frontend implemented on top of turing's e-commerce API. [Shopmate](https://shopmate-six.vercel.app/)

### Features
- User Sign up and Login
- Products
- Product filtering
- Product reviews
- Cart
- Stripe Checkout
